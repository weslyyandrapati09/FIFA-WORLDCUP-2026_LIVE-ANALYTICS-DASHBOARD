⚽ FIFA 2026 Live Dashboard | Power BI

📌 Project Overview

The FIFA 2026 Live Dashboard is an interactive Power BI project built to analyze the latest FIFA World Cup tournament using live tournament data. It transforms raw match statistics into meaningful insights through dynamic visualizations, KPIs, maps, rankings, and interactive filters.

This dashboard enables users to explore team performance, qualification progress, goal statistics, and group-stage outcomes with an intuitive and visually engaging interface.

⸻

🚀 Dashboard Highlights

* 🌍 48 Participating Teams
* ⚽ 60 Total Matches
* 🥅 177 Total Goals
* 📈 Team Qualification Score Analysis
* 🗺️ Geographic Distribution of Teams
* 🏆 Top Performing Teams Ranking
* 📊 Group-wise Performance Analysis
* 🎯 Qualification Bubble Chart
* 🍩 Goal Distribution by Groups
* 🎛️ Dynamic Team & Group Filters
* 📋 Interactive Performance Table

⸻

📊 Key Insights

* 48 teams competed across 12 groups.
* 177 goals were scored during the group stage.
* Qualification scores ranged from 17 to 73.
* Canada secured the highest qualification score.
* Group-level analysis highlights differences in goals, wins, and qualification performance.
* Interactive filters allow users to analyze individual teams or groups instantly.
